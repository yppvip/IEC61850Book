LN / CDC 深度索引

保护、测量/计量、开关控制与自动化的高频模型导读

范围与读法

本页优先覆盖变电站工程中高频出现的 LN 类和 CDC，给出原创导读而非标准逐条复述，也不声称穷尽全部标准 LN。表内“典型 DO / 常见 CDC”用于建立阅读方向；真实模型必须沿 LN@lnType → LNodeType → DOType → DAType / EnumType 核对。

每个 LN 都按六点阅读：功能、典型 DO、常见 CDC、关联角色、工程注意事项、版本/能力提示。DO 名和可选项会随模型版本与 IED 能力变化。

保护类 LN

LN功能典型 DO / 常见 CDC关联角色工程注意事项版本与能力提示
PTOC过流保护相关判据与动作信息。启动/动作、整定与告警类 DO；常见 ACT、ACD、ING/ASG 等。保护 IED、测控及相关跳闸逻辑。核对电流来源、段别/方向、定值组、投退、出口及保护配合；出现 PTOC 不等于已构成完整过流保护方案。具体段数、定值与可选数据依 ICD/一致性声明；Edition 2 / 2.1 与工具支持范围须项目确认。
PDIS距离保护相关功能。测量/阻抗、启动/动作、区段设置类 DO；常见 ACT、ACD、MV、设置类 CDC。线路保护 IED。必须结合电压/电流采样、方向与区段整定、通道配合、跳闸逻辑解释，不能仅凭 LN 名判断保护范围。可用 DO、保护特性和厂商实现差异较大；按 ICD 和项目保护定值单核对。
PDIF差动保护相关功能。差流/制动量、启动/动作与设置类 DO；常见 ACT、ACD、MV、设置类 CDC。主变、母线、线路等差动保护 IED。重点查各侧采样映射、极性/相序、比例制动、CT 配置、通道与出口；单个 PDIF 不说明保护边界已正确。多侧模型及可选项按设备能力不同；尤其要以工程工具生成的实例为准。
PTRC保护跳闸条件处理、出口相关逻辑。跳闸命令/动作状态等 DO；常见 ACT、SPS、INS 等。保护 IED、智能终端、断路器控制链。沿 DataSet、GOOSE、ExtRef、CSWI/XCBR 与一次出口追踪；不要把 PTRC 的状态直接当作断路器已动作反馈。出口数量、跳闸矩阵和测试处理常依设备实现；需结合 CFC/逻辑图和现场试验。

测量与计量类 LN

LN功能典型 DO / 常见 CDC关联角色工程注意事项版本与能力提示
MMXU三相电流、电压、功率、频率等测量建模。电流/电压/功率/频率等 DO；常见 MV、CMV、WYE、DEL。测控 IED、保护 IED、MU 接收侧、网关。连同单位、倍率、质量 q、时标 t 和相序理解；不要把显示值与原始采样或计量值混同。测量项目与可选 DO 依 IED 能力；Edition 变化不自动赋予新增量测能力。
MSQI对称分量、不平衡等测量。序分量/不平衡相关 DO；常见 MV、CMV。保护 IED、测控 IED。确认正/负/零序定义、相序、计算来源与倍率；不可只凭数据名判断算法或保护用途。是否提供各序量、派生量和质量属性以 ICD 为准。
MMTR电能计量相关功能。电能/需量、累计量等 DO；常见计数器、测量及质量相关 CDC。计量 IED、测控/网关。核对正反向、费率、冻结/复位、倍率、结算边界和时间基准；计量数据不等同于实时测量值。计量功能和规约映射可能有项目/厂商差异，应以计量方案和 ICD 确认。

开关控制与自动化类 LN

LN功能典型 DO / 常见 CDC关联角色工程注意事项版本与能力提示
XCBR断路器位置、状态与控制对象相关模型。位置、控制、健康/告警类 DO；常见 DPS、SPC/DPC、SPS。测控 IED、智能终端、保护 IED。分别核对位置反馈、控制路径、机构接口、GOOSE 跳闸及一次设备映射；状态与控制不保证同一信号来源。控制对象、双点编码和可选属性必须从类型模板确认。
CSWI开关操作协调与控制功能。控制、选择、状态和命令相关 DO；常见 SPC/DPC、INC 等。测控 IED、站控应用、智能终端接口。结合 ctlModel、Oper/SBO、操作来源、远方/就地和 XCBR 实际执行链核对。控制服务、命令对象和可选安全模型由 IED 能力决定。
CILO开关操作闭锁相关功能。闭锁允许/原因/状态类 DO；常见 SPS、ENS、ACT 等。测控 IED、保护/联锁逻辑、智能终端。闭锁模型存在不等于闭锁条件已完整；仍需查一次接线、逻辑图、GOOSE 输入、旁路和检修条件。闭锁原因表达及厂家扩展差异常见；私有数据仅标记扩展性质。
RREC自动重合闸相关功能。启动、动作、次数、状态和设置类 DO；常见 ACT、SPS、ING/ASG。线路保护 IED、测控/断路器控制链。核对启动条件、死区、同期/无压逻辑、闭锁、次数、与 PTRC/CSWI/XCBR 的动作链。重合闸策略高度依项目与设备能力，不能按 LN 名推断实际投运方式。
GGIO通用过程输入输出与通用信号建模。厂站通用遥信/遥测/控制 DO；常见 SPS、DPS、MV、SPC 等。测控 IED、智能终端、网关。必须结合 DO 名、Desc、接线图和项目点表解释；不要因 GGIO 出现就猜测其业务用途。GGIO 的实例 DO 命名和使用弹性大；私有补充模型不应被当作通用标准语义。

高频 CDC：从结构语义读到工程检查

CDC典型用途通常关注的属性/语义工程提醒
SPS / DPS单点/双点状态状态值、质量、时标；DPS 的状态编码需追溯类型。双点状态不可简化为任意布尔值；位置反馈还需核对一次设备与采集逻辑。
MV / CMV / WYE / DEL单量、复数量和多相测量数值、质量、时标、单位、倍率、相序。数据集或报告中有数值不代表量纲、质量和采样来源已正确。
ACT / ACD保护动作与方向/复合保护信息相别、方向、动作状态和质量等组合语义。应结合保护逻辑和出口链，不把单个成员当成完整保护结论。
SPC / DPC / APC单点、双点、模拟控制控制模型、操作服务、状态/命令路径。控制可用性还受闭锁、权限、模式、远方/就地与现场安全措施约束。
ING / ASG / VSG整数、模拟、向量设置设置组、编辑值、有效值、单位与范围。不要把设置值或 SCL 中 Val 直接读成运行时当前值。
ENS / ENC枚举状态/枚举设置对应 EnumType 的文字与编码。必须从具体 DA 类型追溯 EnumVal@ord；不要将 XML 属性枚举混为运行数据编码。

版本、能力与扩充边界

Edition 2 / 2.1 的模型演进、可选 DO/DA、工程工具和厂商支持并不完全等同。阅读实际 SCD 时依次确认：项目要求的 Edition、IED 的 ICD/一致性声明、工程工具导入导出结果、LN 实例所指的类型模板以及现场测试记录。新增类别会继续在本页扩充，但不会以“已包含全部标准 LN”的表述替代上述核对。
